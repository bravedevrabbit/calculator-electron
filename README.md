## Main Features

- Simple, Beautiful Interface
- Clean & Well Commented Code
- Cross Platform (Windows, Mac & Linux)

## Guide

```
# Install dependencies

 $ npm install

# Run the app

 $ npm start

# Build for all platforms

 $ npm run build

# Build for one platform ( win / osx / linux ), example for Windows:

 $ npm run win
```
